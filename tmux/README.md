# Tmux 配置目录

这个目录包含了 Tmux 终端复用工具的完整配置和使用指南。

## 📁 文件说明

- **`.tmux.conf`** - Tmux 配置文件（增强版）
- **`tmux-guide.md`** - 完整的使用指南和快捷键速查

## 🚀 快速使用

### 1. 应用配置

```bash
# 复制配置文件到主目录
cp .tmux.conf ~/.tmux.conf

# 如果 tmux 正在运行，重新加载配置
tmux source-file ~/.tmux.conf
```

### 2. 开始使用

```bash
# 创建新会话
tmux new -s mysession

# 在 tmux 内重新加载配置
Ctrl+a r
```

## ✨ 配置特点

- **前缀键**: `Ctrl+a`（更符合人体工程学）
- **分屏**: `Ctrl+a v`（垂直） `Ctrl+a b`（水平），或使用 `Alt+v` / `Alt+b`
- **窗格导航**: 无前缀键快速切换（Alt+h/j/k/l）
- **鼠标支持**: 完全启用
- **美观界面**: 精心设计的状态栏和配色

## 📖 详细文档

查看 [tmux-guide.md](./tmux-guide.md) 获取完整的使用指南、快捷键速查表和实用技巧。

## 🎯 主要改进

相比参考配置，本配置增强了以下内容：

1. **快速导航**: Alt 键无前缀快捷键，提升窗格切换速度
2. **双模式分屏**: 支持前缀键和 Alt 键两种分屏方式
3. **智能分屏**: 新窗格继承当前路径
4. **窗格操作**: 快速旋转（Alt+o）和关闭（Alt+x）窗格
5. **自动编号**: 窗口关闭后自动重新编号
6. **更好的状态栏**: 清晰显示会话、窗口和时间信息
7. **完善注释**: 每个配置项都有清晰说明

## 🔧 自定义

配置文件 `.tmux.conf` 有详细注释，可根据个人喜好修改：

- 修改前缀键
- 调整配色方案
- 添加自定义快捷键
- 集成插件管理器

修改后执行 `Ctrl+a r` 重新加载配置。
