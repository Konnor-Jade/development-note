# Tmux 配置目录

这个目录包含了 Tmux 终端复用工具的完整配置和使用指南。

## 📁 文件说明

- **`.tmux.conf`** - Tmux 配置文件（增强版）
- **`tmux-guide.md`** - 完整的使用指南和快捷键速查

## 🚀 快速使用

### 1. 应用配置

```bash
# 复制配置文件到主目录
cp .tmux.conf ~/.tmux.conf

# 如果 tmux 正在运行，重新加载配置
tmux source-file ~/.tmux.conf
```

### 2. 开始使用

```bash
# 创建新会话
tmux new -s mysession

# 在 tmux 内重新加载配置
Ctrl+a r
```

## ✨ 配置特点

- **前缀键**: `Ctrl+a`（更符合人体工程学）
- **分屏**: `Ctrl+a v`（垂直） `Ctrl+a h`（水平）
- **窗格导航**: Vim 风格（h/j/k/l）
- **鼠标支持**: 完全启用
- **Vi 模式**: 复制模式使用 Vi 快捷键
- **美观界面**: 精心设计的状态栏和配色
